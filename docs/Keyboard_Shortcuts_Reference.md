# CommLink Keyboard Shortcuts - Quick Reference

## Essential Shortcuts

### Core Actions
| Shortcut | Action |
|----------|--------|
| **Ctrl+Return** | Send Message |
| **Ctrl+K** | Connect/Disconnect |
| **Ctrl+R** | Start/Stop Server |

### File Operations
| Shortcut | Action |
|----------|--------|
| **Ctrl+O** | Load Message from File |
| **Ctrl+S** | Save Message to File |

### Message Management
| Shortcut | Action |
|----------|--------|
| **Ctrl+L** | Clear All Messages |
| **Ctrl+E** | Export Messages |
| **Ctrl+Shift+E** | Export Logs |

### Help
| Shortcut | Action |
|----------|--------|
| **F1** | Show This Help |

---

## Pro Tips

### Workflow Acceleration
1. **Quick Connect & Send**: `Ctrl+K` → Type message → `Ctrl+Return`
2. **Load & Send**: `Ctrl+O` → Select file → `Ctrl+Return`
3. **Clear Before Test**: `Ctrl+L` → Send test data

### File Management
- Save frequently used messages: `Ctrl+S`
- Load templates: `Ctrl+O`
- Export results: `Ctrl+E`

### Server Management
- Quick server toggle: `Ctrl+R`
- No need to click buttons - keyboard only!

## Accessibility Features

### Keyboard Navigation
- **Tab**: Move forward through controls
- **Shift+Tab**: Move backward through controls
- **Space/Enter**: Activate focused button
- **Arrow Keys**: Navigate combo boxes and lists

### Screen Reader Support
All controls have descriptive labels for NVDA, JAWS, and Orca screen readers.

### Tooltips
Hover over any control to see:
- What the control does
- Valid input formats
- Keyboard shortcuts
- Best practices

## Display Features

### High DPI Support
CommLink automatically scales for:
- 4K/5K monitors
- High DPI laptop screens
- Multi-monitor setups with different DPIs
